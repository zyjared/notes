# 运算符

- [Using Operators in Your Programs](https://dev.java/learn/language-basics/using-operators/)

:::

<br />

| 运算符               | 优先级                                                          |
| -------------------- | --------------------------------------------------------------- |
| postfix              | `expr++` `expr--`                                               |
| unary                | `++expr` `--expr` `+expr` `-expr` `~` `!`                       |
| multiplicative       | `*` `/` `%`                                                     |
| additive             | `+` `-`                                                         |
| shift                | `<<` `>>` `>>>`                                                 |
| relational           | `<` `>` `<=` `>=` `instanceof`                                  |
| equality             | `==` `!=`                                                       |
| bitwise AND          | `&`                                                             |
| bitwise exclusive OR | `^`                                                             |
| bitwise inclusive OR | `\|`                                                            |
| logical AND          | `&&`                                                            |
| logical OR           | `\|\|`                                                          |
| ternary              | `?` `:`                                                         |
| assignment           | `=` `+=` `-=` `*=` `/=` `%=` `&=` `^=` `\|=` `<<=` `>>=` `>>>=` |

::: info 关于排序

该表格的排序规则：按运算符的优先级从高到低排序。
:::
